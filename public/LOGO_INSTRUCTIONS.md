# Instruksi Menambahkan Logo Hydrangea

## Lokasi File
Simpan gambar logo hydrangea dengan nama: `logo-hydrangea.png` di folder `public/`

## Ukuran yang Disarankan
- Format: PNG dengan transparansi (background transparan)
- Ukuran: 512x512px atau lebih besar (akan di-resize otomatis)
- Format alternatif: SVG juga bisa digunakan (ubah ekstensi menjadi .svg)

## Cara Menambahkan
1. Simpan gambar hydrangea yang ingin digunakan
2. Rename menjadi `logo-hydrangea.png`
3. Copy file ke folder `s_pakar_frontend/public/logo-hydrangea.png`
4. Restart development server jika diperlukan

## Catatan
Logo akan otomatis digunakan di:
- Navbar (App.vue)
- Footer (App.vue)
- Halaman Login
- Halaman Register
- Halaman Reset Password
- Halaman Home
- Halaman Diagnosis
- Dashboard Admin
